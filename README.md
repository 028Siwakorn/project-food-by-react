# แอปพลิเคชันสั่งอาหารอิตาลี

## ภาพรวม
แอปพลิเคชันสั่งอาหารอิตาลีเป็นเว็บแอปพลิเคชันสมัยใหม่ที่ช่วยให้ลูกค้าสามารถดูรายการอาหารและสั่งอาหารอิตาลีแท้ได้ แอปพลิเคชันมีอินเทอร์เฟซที่ใช้งานง่ายสำหรับการดูรายการอาหาร เพิ่มลงตะกร้า และทำการสั่งซื้อ

## คุณสมบัติ
- ดูเมนูพาสต้าและของหวานอิตาลี
- เพิ่มรายการอาหารลงตะกร้า
- ปรับจำนวนรายการในตะกร้า
- ดูสรุปคำสั่งซื้อพร้อมคำนวณภาษีมูลค่าเพิ่ม
- สร้างใบเสร็จรับเงิน
- ออกแบบให้รองรับทั้งมือถือและเดสก์ท็อป

## เทคโนโลยีที่ใช้
- React กับ TypeScript
- Tailwind CSS สำหรับการออกแบบ
- Zustand สำหรับการจัดการสถานะ
- React Router สำหรับการนำทาง

## การติดตั้ง
1. ดาวน์โหลดโปรเจค:
```bash
git clone [repository-url]
```

2. ติดตั้งแพ็คเกจที่จำเป็น:
```bash
npm install
```

3. เริ่มต้นเซิร์ฟเวอร์พัฒนา:
```bash
npm run dev
```

## คู่มือการใช้งาน

### 1. การดูเมนู
- แอปพลิเคชันจะเปิดไปที่หน้าเมนูหลัก
- ใช้แท็บหมวดหมู่ด้านบนเพื่อสลับระหว่าง:
  - พาสต้า (อาหารจานหลัก)
  - ของหวาน
- แต่ละรายการอาหารจะแสดง:
  - รูปภาพอาหาร
  - ชื่ออาหาร
  - ราคา
  - ปุ่มเพิ่มลงตะกร้า

### 2. การเพิ่มรายการลงตะกร้า
1. ดูรายการอาหาร
2. คลิกปุ่ม "+" ที่รายการที่ต้องการเพื่อเพิ่มลงตะกร้า
3. ไอคอนตะกร้าที่แถบนำทางจะแสดงจำนวนรายการ

### 3. การจัดการตะกร้า
1. คลิกไอคอนตะกร้าในแถบนำทางเพื่อดูตะกร้าของคุณ
2. ในหน้าตะกร้า คุณสามารถ:
   - ดูรายการที่เลือกทั้งหมด
   - ปรับจำนวนโดยใช้ปุ่ม "+" และ "-"
   - ลบรายการโดยใช้ปุ่ม "×"
   - ดูยอดรวมย่อย ภาษีมูลค่าเพิ่ม (7%) และยอดรวมทั้งหมด

### 4. การชำระเงิน
1. ตรวจสอบคำสั่งซื้อในตะกร้า
2. คลิกปุ่ม "ชำระเงิน"
3. ระบบจะสร้างใบเสร็จแสดง:
   - รายละเอียดคำสั่งซื้อ
   - รายการอาหาร
   - ยอดรวมรวมภาษีมูลค่าเพิ่ม
   - หมายเลขคำสั่งซื้อ
